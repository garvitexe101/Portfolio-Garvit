## ⚙️ Running Locally

Clone the repository:

```bash
git clone https://github.com/garvitexe101/garvit-portfolio.git
```

Open the folder:

```bash
cd garvit-portfolio
```

Run by simply opening:

```
index.html
```

in your browser.

No build tools or installation required.

---

## 🚀 Deployment

This project can be deployed easily using:

* GitHub Pages
* Netlify
* Vercel

### GitHub Pages

1. Push the repository to GitHub
2. Go to **Settings → Pages**
3. Select branch **main**
4. Save

Your site will be live shortly.
